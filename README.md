# Machine Learning based Automated Resume Shortlisting System

This project is an intelligent system designed to automate the process of resume screening and candidate shortlisting using Natural Language Processing (NLP) and Machine Learning techniques.

It reduces manual effort for recruiters by ranking candidates based on how well their resumes match a given job description.

---

## 🚀 Overview

Recruitment often involves manually reviewing a large number of resumes, which is time-consuming and prone to human error.

This system automates the process by:

* Extracting relevant information from resumes
* Comparing resumes with job descriptions
* Ranking candidates based on similarity scores

👉 The system improves both **efficiency and accuracy** in candidate selection.

## 🏗️ System Architecture

The system follows a structured NLP pipeline:

1. Resume Input
2. Text Preprocessing

   * Tokenization
   * Stopword removal
   * Lemmatization
3. Feature Extraction (TF-IDF)
4. Similarity Calculation
5. Candidate Ranking

👉 The architecture converts unstructured resume data into structured features for analysis. 

---

## ⚙️ Key Techniques Used

### 🔹 Natural Language Processing

* Tokenization
* Stopword removal
* Lemmatization

### 🔹 Feature Engineering

* TF-IDF Vectorization

### 🔹 Similarity Measures

* Cosine Similarity
* Jaccard Similarity
* Dice Similarity
* Overlap Coefficient

These techniques are used to measure how closely a resume matches a job description.

## 📊 Example Workflow

1. Upload resumes and job descriptions
2. System preprocesses text data
3. Converts text into numerical vectors using TF-IDF
4. Computes similarity between resumes and job description
5. Ranks candidates based on scores
6. Displays results with visualizations
